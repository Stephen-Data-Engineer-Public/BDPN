Complete Code Example:

Part 3: Flow Control Statements

Statement Effect Use When Example
continue Skip rest of current iteration Skip invalid items Skip cancelled orders
break Exit entire loop Found what you need Stop when limit reached
pass Do nothing (placeholder) Empty code block TODO sections
else (with loop) Run if no break occurred Verify completion Item not found scenarios

Part 4: Common Patterns

Pattern: Filtering

valid_orders = [] for order in orders: if order["status"] == "active": valid_orders.append(order)

Pattern: Aggregation

total = 0 count = 0 for item in items: if item["price"] > 0: total += item["price"] count += 1

Pattern: Search & Stop

found = None for item in inventory: if item["id"] == target_id: found = item break

Pattern: Skip Invalid

for record in data: if not is_valid(record): continue process(record)

‚úÖ Clean & Clear

‚ö†Ô∏è Common Mistakes to Avoid

  • Assignment vs Comparison: Use == not = in conditions
  • Infinite Loops: Always update loop condition in while loops
  • Break in Nested Loops: Only exits innermost loop
  • Modifying List While Looping: Can cause unexpected behavior
